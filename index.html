ИИ-подбор по протоколам (Gemini, локальный сервер)

Локально: страница + rag_server на http://localhost (или 127.0.0.1), ключ в .env. На хостинге (например Render) API и страница с одного адреса — поле «URL API» подставится само (тот же домен). GitHub Pages без бэкенда к вашему ПК не подключится.
