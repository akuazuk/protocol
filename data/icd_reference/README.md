# Справочники МКБ-10 для сопоставления запросов и поиска по кодам

## 1. WHO ICD-10 2016 (англ.), табличный meta

- **Источник:** [WHO ICD-10 Downloads](https://icdcdn.who.int/icd10/index.html) → *2016* → *Plain text tabular format* → `icd102016enMeta.zip`
- **Файлы в репозитории:** `icd102016enMeta.zip` (архив), распаковано в `who_icd10_2016_en/`
- **Назначение:** формальные коды и английские названия; структура совпадает с международной МКБ-10, используемой в РБ.
- **Ограничение WHO (readme в архиве):** meta **не заменяет** полное кодирование по правилам томов 1–3.

Сборка JSON для быстрого поиска по коду:

```bash
python3 scripts/icd10_who_txt_to_json.py
```

Результат: `icd10_who_2016_terminal_codes.json` (массив объектов `code`, `title_en`, `terminal`, `level`).

## 2. Русскоязычный справочник (Excel)

- **Файл:** `mkb10_ru_mkb10su.xlsx`
- **Скачан с:** https://mkb10.su/download/mkb10.xlsx (сторонний ресурс; коды МКБ-10 совпадают с международными, формулировки на русском — удобно для сопоставления с текстом запроса).
- **Лицензия / актуальность:** уточняйте на источнике; для юридически значимых решений ориентируйтесь на официальные издания.

## 3. Минздрав Республики Беларусь (официально)

- **Страница:** [Справочник МКБ-10 (электронная форма)](https://minzdrav.gov.by/ru/dlya-spetsialistov/klassifikatory/spravochnik-mezhdunarodnoy-statisticheskoy-klassifikatsii-bolezney-i-problem-svyazannykh-so-zdorovem.php)
- На странице указан архив для АИС; прямая ссылка в HTML может устареть (**при проверке в 2026-04 ссылка на zip давала 404**). При необходимости скачайте актуальный файл с сайта вручную и положите сюда, например `spr_mkb10_minzdrav_rb.zip`.

## Идея для бэкенда

1. Извлекать из текста запроса кандидаты кодов вида `J20`, `J20.9`, `A00.0` (регулярное выражение по шаблону МКБ-10).
2. Сверять с `icd10_who_2016_terminal_codes.json` или с таблицей из Excel.
3. Искать протоколы, в которых встречаются те же коды (индекс / полнотекст / метаданные PDF).
