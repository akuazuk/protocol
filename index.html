Поиск протокола

Введите симптомы, диагноз или ключевые слова на русском — например: астма, боль в груди, сахарный диабет, язва желудка.

Данные собраны из раздела «Клинические протоколы» на сайте Министерства здравоохранения Республики Беларусь. Логотип: Кравира.

Как работает поиск: по тексту внутри протоколов (файл corpus.json), по названию и рубрике, по словарю подсказок симптомов; при наличии embeddings.json — дополнительно семантическое сходство (модель multilingual-e5-small в браузере для запроса). Облачный RAG (например Vectara) возможен через свой backend с API-ключом — ключ нельзя вставлять в статический сайт.

Сборка данных: python3 extract_corpus.py, затем опционально python3 build_semantic_embeddings.py (см. requirements-search.txt). Сайт: GitHub Pages или python3 -m http.server 8080.
