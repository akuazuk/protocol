Можно выбрать несколько областей — поиск будет сильнее ориентироваться на протоколы из этих разделов каталога.

На сайте Минздрава нет отдельной рубрики «Терапия»: перечень совпадает с каталогом (24 раздела). Общие и «терапевтические» протоколы отнесены к профильным разделам — например, болезни системы кровообращения, гастроэнтерология, пульмонология, эндокринология, неврология и др.

Поиск протоколов

Сначала при необходимости заполните контекст и рубрики выше, затем введите запрос и нажмите «Найти протоколы».

Можно указать диагноз или код МКБ-10, динамику и лечение. Подбор запускается после нажатия «Найти протоколы» — до этого запрос только вводится в поле.

На смартфоне разрешите доступ к микрофону. Распознавание — в браузере, аудио на сервер не отправляется.

Как устроен поиск

Текст запроса сопоставляется с фрагментами официальных PDF клинических протоколов Минздрава РБ; сервис предлагает релевантные документы и краткое пояснение. Это ориентир в документах, не диагноз и не замена очного приёма.

1
При необходимости заполните контекст пациента и рубрики (блок выше), затем введите запрос и нажмите «Найти протоколы» — так подбор точнее.
2
В «Результат подбора» — список протоколов с оценкой близости (0–100%): технический ориентир для выбора PDF, не «вероятность диагноза». Даётся краткий разбор по запросу.
3
По желанию: дифференциальные гипотезы (кнопка «Искать протокол» подставляет выбранную строку в запрос) и шаблон консультативного заключения по развёрнутой выдержке — только черновик; нужна проверка и правка врачом.

Примеры формулировок

«сухой кашель около двух недель» «изжога после еды» сахарный диабет 2 типа код J20.9

Совет: на телефоне можно надиктовать жалобы кнопкой «Голос» — текст остаётся в браузере.
